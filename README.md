# SmartMail AI

> AI-Powered Intelligent Email Management System

SmartMail AI is an AI-powered email management platform that helps users organize, summarize, classify, and analyze emails intelligently. It combines Machine Learning and Natural Language Processing (NLP) to improve productivity by reducing email overload.

---

##  Features

### Authentication
- User Registration
- User Login
- JWT Authentication
- Secure Logout

### Email Management
- Gmail Integration
- Fetch Emails
- Search Emails
- Filter Emails
- Email Categories

###  AI Features
- Email Summarization
- Spam Detection
- Priority Detection
- Smart Email Categorization
- AI Insights

### Dashboard
- Email Analytics
- Daily Activity
- Spam Statistics
- Category Distribution
- Productivity Dashboard

##  Installation

### Clone Repository

```bash
git clone https://github.com/JogaEkshitha/SmartMail-AI.git
```

### Backend

```bash
cd backend

python -m venv venv

source venv/bin/activate

pip install -r requirements.txt

python manage.py migrate

python manage.py runserver
```

### Frontend

```bash
cd frontend

npm install

npm run dev
```
